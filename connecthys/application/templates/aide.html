Renseignements

La page Renseignements vous permet de visualiser les renseignements sur les membres de votre famille. Cliquez sur le bouton Modifier pour envoyer une demande de modification de renseignements à l'administrateur du portail.
{% endif %} {%if GetParametre("ACTIVITES_AFFICHER", dict_parametres) == 'True' %}

Activités

La page Activités liste les inscriptions de chaque membre de la famille. Il est possible de demander l'inscription à une activité en cliquant sur le bouton Inscrire à une activité. Vous devrez alors attendre que votre demande soit traitée et validée par l'administrateur du portail.
{% endif %} {%if GetParametre("RESERVATIONS_AFFICHER", dict_parametres) == 'True' %}

Réservations

La page Réservations liste les périodes de réservations disponibles pour chaque membre de la famille inscrit à une activité. Si vous souhaitez saisir, modifier ou supprimer des réservations, cliquez sur l'activités puis sur la période désirée. Le calendrier de la période apparaît pour vous permettre de cocher ou décocher des réservations. Cliquez sur le bouton Envoyer la demande puis patientez jusqu'au traitement de votre demande par l'administrateur du portail. Cliquez sur le lien Revenir à la liste des périodes pour accéder au calendrier d'une autre période ou d'un autre membre de la famille. Après traitement, vos réservations validées sont colorées en vert alors que les réservations refusées (pour manque de place par exemple) seront colorées en rouge.
{% endif %} {%if GetParametre("FACTURES_AFFICHER", dict_parametres) == 'True' %}

Factures

La page Factures liste les dernières factures de la famille et permet d'identifier rapidement les éventuelles factures impayées. Si vous souhaitez recevoir un duplicata, cliquez sur l'image sur la ligne correspondante puis sélectionnez le moyen de transmission souhaité. Vous devrez alors attendre que votre demande soit traitée et validée par l'administrateur du portail.
{% endif %} {%if GetParametre("REGLEMENTS_AFFICHER", dict_parametres) == 'True' %}

Règlements

La page Règlements liste les derniers règlements de la famille et propose de visualiser ceux qui n'ont pas encore été encaissés. Si vous souhaitez recevoir un reçu de règlement, cliquez sur l'image sur la ligne correspondante puis sélectionnez le moyen de transmission souhaité. Vous devrez alors attendre que votre demande soit traitée et validée par l'administrateur du portail.
{% endif %} {%if GetParametre("PIECES_AFFICHER", dict_parametres) == 'True' %}

Pièces

La page Pièces liste les pièces manquantes à votre dossier. Si l'option Téléchargement de pièces est activée, vous pouvez cliquer sur le bouton d'une pièce pour la télécharger ou l'imprimer afin de la remplir avant de la remettre à l'administrateur du portail.
{% endif %} {%if GetParametre("COTISATIONS_AFFICHER", dict_parametres) == 'True' %}

Cotisations

La page Cotisations liste la ou les cotisations manquantes à votre dossier. Vous devez alors contacter l'administrateur du portail afin de régulariser votre dossier.
{% endif %} {%if GetParametre("HISTORIQUE_AFFICHER", dict_parametres) == 'True' %}

Historique

La page Historique liste les dernières demandes que vous avez effectué sur le portail. L'historique vous permet de suivre le traitement de chaque demande. Pour certaines demandes, il est possible de cliquer sur le bouton Détail pour accéder aux détails de la demande. Si une demande n'a toujours pas été transmise à l'administrateur du portail, il est possible de la supprimer en cliquant sur le bouton Supprimer.
{% endif %}
